Plum
=========

Plum is a unique, ultra creative multipurpose WordPress theme with complete support for WooCommerce. Plum supports all basic features and is fully mobile friendly. And with Support for Custom Widgets, Slider and Featured Posts section, etc you can make this theme look even more beautiful on your website. 
Demo Here: http://demo.inkhive.com/plum/

Getting Started
---------------

How to Install the Theme?

1. Go to WP Admin > Appearance > Themes > Add New.
2. Search for "Plum".
3. Click on Install and Activate.

Alternatively,

1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the plum.zip file, in which you found this README.txt
3. Click on Install and Activate.

Setting it Up
-------------

This theme is built using Customizer. So to Configure theme settings like Featured Content Area, Sliders, Social Icons, etc from 
	
	Dashboard > Appearance > Customizer
